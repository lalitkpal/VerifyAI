Verification Rules Index

Details and configuration of the active verification engines

Code Syntax parser

Validates programming blocks extracted from Markdown. Python scripts are parsed via AST compilation to ensure zero syntax errors. JSON blocks are verified against RFC 8259 parser logic.

Secret Scanner

Scans code blocks for hardcoded cloud identifiers. Checks for high-entropy strings, AWS API access keys, secret keys, and general tokens (Slack, GitHub, private keys) before publication.

Completeness Audit

Checks for incomplete functions or lazy coding placeholders. Identifies markers such as "// TODO", "# implement code here", or ellipses ("...") left inside functions.

Citation Web-Verifier

Extricates citations from Perplexity/Gemini outputs. Executes concurrent, lightweight HTTP connection and status calls (HEAD/GET requests) to ensure external reference links aren't dead or hallucinated.

Jailbreak/PII Shield

Analyzes prompt contexts for override triggers and guards outputs against credit card patterns, social security numbers, emails, and phone number leaks.
